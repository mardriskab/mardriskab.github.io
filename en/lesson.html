STEP BY STEP

How to use the MardRisk AI app

1

1. Open the chat

Open the MardRisk app or web chat. You will see an input field where you can type commands.

2. Type a command

Start the command with the “/” symbol (for example: /analyze BTC or /RiskScore).

The AI will analyze your query, display charts, forecasts, and practical advice.
